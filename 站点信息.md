  - `活动核价.py` 中的 `PRICE_MIN` / `PRICE_CAP` 与上表一致

- 价格变更需要三处同步更新：`活动核价.py` + `价格表.md` + 本文

## 站点选择操作说明

- 组件：Beast UI Select 多选组件（`ST_dropdownPanel` 内 108 个 LI）
- 操作：打开下拉 → 读取可选站点 → 匹配目标 → 逐个勾选 → 安全检查 → 关闭
- 勾选：点击 `[data-testid="beast-core-checkbox-checkIcon"]`
- 验证：`data-checked="true"`
- 安全检查：取消不在白名单的已选项
- 关闭：点击 select header 切换关闭
